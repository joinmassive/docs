## Usage data

**{companyName}'s** usage endpoint returns aggregated usage data for your account. The endpoint
accepts `GET` requests at **https://&zwj;{apiEndpoint}/usage**.

By default, usage data is returned for the past 7 days. You can specify a custom date range, filter
by service, disaggregate by time period, or filter by email.

### Report parameters

Request parameters can be added in a
[standard query string](https://developer.mozilla.org/en-US/docs/Learn/Common_questions/What_is_a_URL#parameters).
The keys and values **{companyName}** supports are as follows:

| Key       | Required | Value                                                                                                                                                   |
| :-------- | :------- | :------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `from`    | ⬜       | The start date of the report in `YYYY-MM-DD` format (UTC); defaults to 7 days ago                                                                       |
| `to`      | ⬜       | The end date of the report in `YYYY-MM-DD` format (UTC); defaults to today                                                                               |
| `service` | ⬜       | The service to filter by, `browser` or `search`; all services are included by default                                                                    |
| `by`      | ⬜       | The time period to disaggregate by, `day`, `week`, or `month`; data is aggregated across the full date range by default                                  |
| `email`   | ⬜       | The email address of the account to filter by; all accounts are included by default                                                                      |

### Response

The response is a JSON array of usage records:

| Key           | Value                                                                           |
| :------------ | :------------------------------------------------------------------------------ |
| `service`     | The name of the service (e.g., `browser`, `search`)                             |
| `date`        | The date or start date of the period (if disaggregated)                         |
| `requests`    | The total number of API requests                                                |
| `successes`   | The number of successful requests                                               |
| `credits`     | The number of credits consumed                                                  |

### Examples

#### Date range

To retrieve usage data for a specific date range:

```bash
$ curl -H "Authorization: Bearer $MASSIVE_TOKEN" \
  'https://unblocker.joinmassive.com/usage'\
  '?from=2025-01-01&to=2025-01-31'
```

#### Service filtering

To filter usage data by a specific service:

```bash
$ curl -H "Authorization: Bearer $MASSIVE_TOKEN" \
  'https://unblocker.joinmassive.com/usage'\
  '?service=browser'
```

#### Time disaggregation

To disaggregate usage data by day:

```bash
$ curl -H "Authorization: Bearer $MASSIVE_TOKEN" \
  'https://unblocker.joinmassive.com/usage'\
  '?by=day&from=2025-01-01&to=2025-01-31'
```

#### Email filtering

To filter usage data by account email:

```bash
$ curl -H "Authorization: Bearer $MASSIVE_TOKEN" \
  'https://unblocker.joinmassive.com/usage'\
  '?email=user@example.com'
```
