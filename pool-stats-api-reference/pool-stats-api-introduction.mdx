# Key Features

- Get live node pool statistics with filtering
- Group results by various dimensions (country, city, ASN, pool type, ZIP)
- Retrieve available server regions

# Getting Started

To start using the Pool Stats API, you'll need:

1. An API key (contact Massive support to enable it for this API)
2. Basic understanding of RESTful APIs and HTTP methods

# API Endpoints

The base URL for all API requests is:

https://api-network.joinmassive.com/pool-stats

For detailed information on available endpoints and operations, see the API Reference:

- [Get Live Stats](/pool-stats-api-reference/get-live-stats) - Retrieve real-time node pool statistics
- [Get Available Regions](/pool-stats-api-reference/get-available-regions) - List available server regions

# Authentication

All API requests must include your API key in the `x-api-key` header. Learn more in our [Authentication Guide](/pool-stats-api-reference/pool-stats-api-authentication).
